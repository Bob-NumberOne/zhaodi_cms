表名：mode_triggerworkflowset
中文名称：数据审批基本信息
表类型：
返回目录
说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 id id int
2 模块名称 modeid int 模块id，对应modeinfo表的id
3 被触发流程类型 workflowid int 工作流程id，对应workflow_base中的id，只有自定义的新表单的流程才能作为被触发的流程
4 被触发流程创建人 wfcreater int 被触发流程创建人
1:模块当前操作人
2:模块创建人
3:模块人力资源相关字段
5 模块人力资源相关字段 wfcreaterfieldid int 被触发流程创建人，从表单建模字段中取值时对应的字段id，对应workflow_billfield表的id
6 流程触发成功回写 successwriteback varchar 4000 触发审批流成功或者失败的时候，可以设置回写值，用来修改当前模块某些主字段的值，比如:a='2'，
如果要修改多个字段的值，请用","将多个字段的值隔开，比如a='2',b='3',c='abc'，
其中a,b,c是指表单中数据库字段名。
7 流程触发失败回写 failwriteback varchar 4000 同上
8 流程触发条件 showcondition text 流程触发条件sql
9 流程触发条件显示名称 showconditioncn text 流程触发条件sql对应的中文显示名称

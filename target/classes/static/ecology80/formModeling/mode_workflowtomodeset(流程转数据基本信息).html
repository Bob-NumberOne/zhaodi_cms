表名：mode_workflowtomodeset
中文名称：流程转数据基本信息
表类型：
返回目录
说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 id id int
2 模块id modeid int 对应modeinfo表的id
3 流程类型 workflowid int 工作流程id，对应workflow_base表的id
4 模块创建人 modecreater int 1：流程当前操作人
2：流程创建人
3：流程人力资源相关字段
5 流程人力资源相关字段 modecreaterfieldid int 当使用流程中的字段的值作为模块的创建人时字段的id，对应workflow_billfield表的id
6 触发节点 triggerNodeId int 对应workflow_nodebase表的id
7 触发时间 triggerType int 0：离开节点
1：到达节点
8 是否启用 isenable char 1：启用
0：不启用
只有启用时，流程转数据才能生效
9 触发表单 formtype varchar 1000 maintable：主表
detail1：明细1，依此类推
10 接口id actionid int 对应在流程中设置的节点预处理和后处理的接口id,对应workflowactionset表的id
11 主表操作类型 maintableopttype varchar 1 1：插入
2：更新
12 主表更新条件 maintableupdatecondition varchar 256

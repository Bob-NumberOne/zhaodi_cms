表名：mobileappbaseinfo
中文名称：应用信息表
表类型：
返回目录
说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 id int N Y Y
2 应用名称 appname varchar 1000 应用的名称
3 图标路径 picpath varchar 1024 应用对应的图标路径，如/mobilemode/images/a.png
4 描述 descriptions varchar 1000 应用的描述信息
5 显示顺序 showorder int 显示顺序，升序排序，越小的值排序越靠前
6 是否删除 isdelete int 是否删除，1是，0否
7 表单id formId int 无用，已废弃
8 是否发布 ispublish int 应用发布状态：1发布，2下架
9 模块id modelid int 无用，已废弃
10 发布id publishid varchar 1000 发布到Emobile配置文件中对应component的id
11 应用是否包含头部 hasHeader varchar 10 为1时包含头部，为空或者0则不包含头部
12 皮肤 skin varchar 500 存放该应用所引用的皮肤id，该id为32位的uuid

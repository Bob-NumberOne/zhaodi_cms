表名：appformui
中文名称：模块UI(布局，列表)对应表
表类型：
返回目录
说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 id int N Y Y
2 表单id formid int 布局对应的表单id，workflow_bill表的id列外键
3 UI内容 uiContent text 存放UI布局的html代码
4 UI类型 uiType int UI类型：0新建,1显示,2编辑,3列表
5 应用id appid int 所属应用，mobileAppBaseInfo表的id列外键
6 UI名称 uiname varchar 1000 ui的中文名称，根据ui类型分别对应表单建模模块下的新建布局名称，编辑布局名称，显示布局名称，查询列表的名称
7 是否删除 isdelete int 是否删除，1是，0否
7 是否隐藏 isHide varchar 是否隐藏，1是，NULL否
8 UI模板 uiTemplate text 存放UI布局的html代码
9 默认名称 defaultTitle varchar 1000 无用，已废弃
10 模块id entityId int 模块id，mobileAppModelInfo表id列的外键
11 查询id sourceid int 表单建模，查询列表id。mode_customsearch表id列的外键
12 设备类型id mobiledeviceid int 设备类型，mobiledevice表id列的外键
13 父id parentid int 父id，和本表id列自身关联

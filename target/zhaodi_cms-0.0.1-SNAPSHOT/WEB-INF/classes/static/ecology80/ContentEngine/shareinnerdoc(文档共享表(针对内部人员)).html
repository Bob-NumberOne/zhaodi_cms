说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 ID id int N Y Y 　
2 表示的是不同文档的ID sourceid int N 　
3 共享记录的类型 type int N 1.人员，2.分部，3.部门，5.所有人，6.群组
4 如果当共享的类型为角色的时候. 其值表于为角色ID角色级别ID content int N (0: 表部门 1：表分部 2：表总部)格式如 230其中最后一位表示角色级别ID
5 表示此项不需要最低安全级别 seclevel int N 　
6 共享级别 sharelevel int N 1.查看，2.编辑，3.完全控制
7 指此共享记录的来源 srcfrom int N 1. 人力资源，2.分部，3.部门，4.角色，5.所有人，80.创建人本身，81.创建人直接上级，84.创建人同分部，85.创建人同部门，-81.创建人是客户时创建人的经理，86.文档有者
8 相关人员 opuser int N 指此共享记录的原始所有者,一般都与srcfrom合起来用比如说 srcfrom 为81 opuser为67 就是指 67这个人的直接上级
9 是否来自于流程 sharesource int 1: 表示从流程那边过来，0：表示非流程部分过来
10 下载权限级别 downloadlevel int 0：不可下载，1：可下载
11 岗位级别 joblevel char(10)
12 指定部门 jobdepartment char(10)
13 指定分部 jobsubcompany char(10)
14 岗位id jobids char(10)

表名：workflow_subwfset
中文名称：子流程设置表
表类型：
返回目录
说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 子流程设置表id id int N 由SequenceIndex表得到，对应的indexdesc为Workflow_SubwfSetId
2 主流程id mainWorkflowId int 　
3 子流程id subWorkflowId int 　
4 触发节点id triggerNodeId int
5 触发时间 triggerTime char 1 1:到达节点,2:离开节点
6 子流程创建人类型 subwfCreatorType char 1 1:主流程当前操作人,2:主流程创建人,3:主流程单人力资源字段
7 主流程单人力资源字段id或多人力资源字段id或收文单位字段id subwfCreatorFieldId int 　
8 流程之间的意见是否可相互查看 isread int 0 0否，1是
9 触发类型 triggerType char 1 1：自动触发，2：手动触发
10 触发操作 TriggerOperation char 1 1：批准
2：退回
""或其他：任何操作都触发
11 触发源 triggerSource int 　
12 触发源类型 triggerSourceType varchar 1000 　
13 触发源顺序 triggerSourceOrder int 　
14 触发条件 triggerCondition varchar 1000 　
15 是否为已查看节点 isreadNodes varchar 1000 　
16 是否为已看出主流程 isreadMainwf char 1 　
17 是否为已看出主流程节点 isreadMainWfNodes varchar 1000 　
18 是否是已查看并行流程 isreadParallelwf char 1 　
19 是否是已查看并行流程节点 isreadParallelwfNodes varchar 1000 　
20 是否使能 enable char 1 　
21 是否允许创建节点 isStopCreaterNode char 1

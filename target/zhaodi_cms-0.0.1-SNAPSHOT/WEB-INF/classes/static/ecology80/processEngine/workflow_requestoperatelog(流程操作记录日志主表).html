表名：workflow_requestoperatelog
中文名称：流程操作日志表（非签字意见）
表类型： 系统表
返回目录
说明：用来记录用户的操作， 作为强制收回时的依据。
本表为操作日志主表， 与操作日志详细表为1：N关系， 当本条记录被强制收回时，会将isinvalid置为1，并记录强制收回人的id和操作日期， 然后还原明细表中记录的本次操作的数据

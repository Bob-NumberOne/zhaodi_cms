表名：apphomepage
中文名称：自定义页面表
表类型：
返回目录
说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 id int N Y Y
2 应用id appid int 所属应用，mobileAppBaseInfo表的id列外键
3 页面内容 pagecontent text 存放页面html内容
4 设备类型id mobiledeviceid int 设备类型，mobiledevice表id列外键
5 父id parentid int 父id，和本表id列自身关联
6 页面名称 pagename varchar 1000 自定义页面的中文名称
7 页面描述 pagedesc varchar 1000 自定义页面描述信息
8 是否首页 ishomepage int 是否是首页，1为是，0或者空为否
9 页面属性 pageattr text 页面本身的一些属性设置，json数据格式
10 顺序 orderid int 显示顺序，升序排序，越小的值排序越靠前

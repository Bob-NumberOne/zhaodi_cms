说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 id int N Y Y
2 设备名称 devicename varchar 1000 设备名称，Mobile：手机，Pad：平板
3 图片路径 picpath varchar 1024 图片路径，用于定义移动建模自定义页面设计区域的背景图
4 宽度 width int 宽度，用于定义移动建模自定义页面设计区域的宽度
5 高度 height int 高度，用于定义移动建模自定义页面设计区域的高度
6 默认启用 defaultenable int 默认启用：1：默认启用，0：不启用。新建自定义页面会使用默认启用的设备类型作为初始布局

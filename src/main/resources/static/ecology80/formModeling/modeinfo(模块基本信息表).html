表名：modeinfo
中文名称：模块基本信息表
表类型：
返回目录
说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 模块id id int N Y Y
2 名称 modename varchar 1000
3 描述 modedesc varchar 1200
4 所属应用 modetype int 对应表modetreefield中的id
5 表单 formid int 对应表workflow_bill中的id,表单建模中使用的表单一般都是自定义表单，formid的值一般都是小于0的整数
6 附件上传1级目录 maincategory int 附件上传目录，字段maincategory、subcategory和seccategory结合使用，共同组成附件上传目录
7 附件上传2级目录 subcategory int 同上
8 附件上传3级目录 seccategory int 同上
9 允许创建时导入明细 isimportdetail int 勾选此选项可以在新建时，使用Excel导入明细
10 codeid codeid int 编码ID(暂保留)
11 custompage custompage varchar 1000 自定义页面地址，二次开发使用
12 是否允许修改默认共享 defaultshared char 1 0：不允许
1：允许
允许时，有完全控制权限的用户可以新增或删除默认共享
13 是否允许修改非默认共享 nondefaultshared char 1 0：不允许
1：允许
允许时，有完全控制权限的用户可以新增或删除非默认共享
14 显示顺序 dsporder float
15 modecode modecode varchar 1000 32位UUID，导入模块时的唯一识别判断
16 是否逻辑删除 isdelete int 0：未删除
1：已删除
已删除的模块不会显示出来
17 所属分部 subcompanyid int 对应表HrmSubCompany中的id字段，用于表单建模分权功能
18 是否允许回复 isallowreply int 0：不允许回复
1：允许回复
18 附件上传目录类型 categorytype int 0：固定目录
1：选择目录
18 选择框字段id selectcategory int

中文名称：文档附件图片表
表类型：
返回目录
说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 图片附件id id int N 　
2 对应文档id docid int 　
3 图片文件id imagefileid int 由SequenceIndex表得到，和ImageFile表相关联
4 文件名称,包括后缀名 imagefilename varchar 1000 　
5 文件描述 imagefiledesc varchar 1000 由用户给出
6 文件宽度 imagefilewidth int 象素 只对文档中的图片文件,不包括附件及附件中的图片文件
7 文件高度 imagefileheight int 象素：只对文档中的图片文件,不包括附件及附件中的图片文件
8 文件大小 imagefielsize int 　
9 文件类型 docfiletype varchar 1000 1:文档中的图片
2:附件(包括附件中的图片)
3.World 文档
4.Excel 文档
5.PPT文档
6.金山WPS文档
7、docx文档
8、xlsx文档
9、pptx文档
10、金山ET文档
11、html文档中的视频
10 唯一标识 versionId int 也用来区别World或Excel文档的版本
11 World或Excel文档版本描述 versionDetail varchar 1000 　
12 的 isextfile char 1 1 　
13 是否使用模板 hasUsedTemplet char 1 0 1：使用，其他不使用

表名：hpinfo
中文名称：首页信息表
表类型：
返回目录
说明：
序号 中文名称 英文名称 数据类型 长度 允许空值 主键 默认值 自增 说明
1 ID id int N Y 　
2 首页名称 infoname varchar 1000 　
3 首页描述 infodesc varchar 1000 　
4 样式ID styleid varchar 1000 　
5 布局ID layoutid int N 　
6 所属分部ID Subcompanyid int N 为0表示所有的分部都要用
7 是否启用 isUse char 1 N 0.不启用 1.启用 2.表示启用并且不能修改
8 创建者类型 creatortype int 1:内部用户ID 2外部用户ID 3:分部ID 4:系统管理员
9 创建者ID creatorid int 　
10 显示类型 showtype int 1 0:不确定 1:一级首页 2:二级首页
11 二级首页的主页 parentHpid int 0 如果是一级首页此处则为0
12 首页排序须序 ordernum varchar 1000 　
13 是否锁定 isLocked int 0 1:锁定 0:非锁定
14 首页菜单样式ID menustyleid varchar 1000 　
15 首页维护者ID maintainer varchar 1000 　
16 首页语言信息 hplanuageid int 7 　
17 首页创建者ID hpcreatorid int 1 　
18 最后修改日期 hplastdate varchar 1000 　
19 最后修改时间 hplasttime varchar 1000 　
20 上级首页ID pid int 　
21 排序字段 ordernum1 int
